# frontend for tvoy-quest

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

<p>Приложение для проведения онлайн квеста на мобильных устройствах</p>
<p>Технологии frontend: Vue, vuex, vue-router</p>
